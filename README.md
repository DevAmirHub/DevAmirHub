## 🚀 About Me  

I'm a **Backend Developer** with strong expertise in **Laravel**, **RESTful APIs**, scalable architecture, and backend infrastructure.  
My primary focus is building backend systems that are:

- scalable  
- maintainable  
- cleanly architected  
- production‑ready  

I design backend applications with structured architecture, modularity, and performance in mind. Most of my work includes:

- API design & backend architecture  
- authentication / authorization systems  
- business logic implementation  
- database schema design & optimization  

I also work with **WordPress (custom plugins, theme development, and headless setups)** when needed, always applying backend principles (modularity, architecture, performance).

Though backend is my focus, I have hands‑on experience with **React, Next.js, Livewire 3, TailwindCSS**, and even **Three.js / GSAP**, which helps me collaborate across the stack and ship complete end‑to‑end solutions.
